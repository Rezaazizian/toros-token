# TRS Staking (Anchor)

Minimal scaffold for a staking program on Solana Devnet.

## Prerequisites
- Rust + cargo
- Solana CLI
- Anchor CLI (`cargo install --git https://github.com/coral-xyz/anchor anchor-cli --locked`)

## Build
```bash
anchor build
```

## Deploy (devnet)
```bash
anchor deploy
```

Config: see `Anchor.toml`.
